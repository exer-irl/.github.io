üè≠ IoT-Based Press Monitoring & Optimization System

IoT Raspberry Pi Industrial Engineering Python

Challenge

Manufacturing facility with 12 hydraulic presses had zero visibility into machine performance. Production managers relied on manual logbooks, resulting in unknown downtime causes, frequent overheating issues, inconsistent cure times, and reactive maintenance costing $800K quarterly in lost production.

Solution: Raspberry Pi IoT Implementation

  • Deployed Raspberry Pi 4 computers to each of 12 presses with custom sensor integration
  • Real-time metrics tracking:
    • Downtime events with automatic categorization (mechanical, material, changeover)
    • Temperature monitoring (hydraulic fluid & press platens) with alert thresholds
    • Bump count and force measurement for quality control
    • Cure time tracking with automatic variance detection
    • Cycle time, setup time, and OEE calculations
  • Built Python-based data collection system with MQTT protocol for low-latency communication
  • Developed real-time dashboard using Grafana with alerts sent to production managers
  • Implemented SQLite edge computing for local data buffering during network issues
  • Created ETL pipeline to PostgreSQL database for historical analysis and reporting

Key Innovation
